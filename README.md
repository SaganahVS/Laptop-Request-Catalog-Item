# Laptop-Request-Catalog-Item


## Overview

The **Laptop Request Catalog Item** automates the process of requesting laptops in **ServiceNow**. It simplifies user requests, automates approvals, and ensures accurate laptop assignments, improving IT service efficiency.


## Key Features

* Simple and user-friendly request form
* Automatic approval and fulfillment workflow
* Email notifications for each request stage
* Laptop assignment tracking through CMDB
* Business rules to prevent duplicate or invalid requests


## Components

* **Catalog Item** – Laptop request form
* **Flow Designer** – Handles approvals and fulfillment
* **Business Rules** – Validates and controls request logic
* **Notifications** – Alerts users and approvers


##  Workflow

1. User submits laptop request
2. Manager approval is triggered automatically
3. IT fulfills and assigns the laptop
4. User gets confirmation email


##  Tools Used

* ServiceNow Platform
* Flow Designer
* Business Rules (JavaScript)
* CMDB Integration
